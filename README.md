## 📌 Project Overview

This repository contains mini-project implementations for the Artificial Neural Network (ANN) course. The project focuses on understanding and applying fundamental neural network models through practical implementation.

---

## 🚀 Projects Included

### 🔹 1. Perceptron Model

* Implemented a basic perceptron using Python
* Tested on logical operations (AND gate)
* Demonstrates binary classification

---

### 🔹 2. Competitive Learning Network (Unique)

* Implemented a shape-based competitive learning model
* Uses noisy circle and square data
* Neurons learn to specialize in identifying different shapes
* Demonstrates unsupervised learning and pattern grouping

---

### 🔹 3. Feedforward Neural Network (Unique)

* Built a neural network to predict student performance
* Inputs: Study hours, sleep hours, attendance
* Output: Pass / Fail prediction
* Demonstrates real-world application of ANN

---

## 📊 Results
